# Shop (Vite + React)

Proyecto creado automáticamente. Contiene una pantalla de carga de 5 segundos y la página principal con "Hola Mundo".

Cómo ejecutar en desarrollo:

```bash
cd /workspaces/shop
npm install
npm run dev
```

Build de producción:

```bash
npm run build
npm run preview
```

Archivos importantes:

- `index.html` - HTML de entrada
- `src/main.jsx` - punto de entrada React
- `src/Loading.jsx` - componente que muestra la pantalla de carga (5s) y luego muestra "Hola Mundo"
- `src/App.jsx` - componente raíz
